# NOOP v8.7.1 (fork)

This fork release merges upstream NOOP 8.7.0 - see the
[upstream 8.7.0 notes](https://github.com/ryanbr/noop/releases/tag/v8.7.0) for the full breakdown of
the sync chip on Today (#245), the future-dated strap clock warning (#324), the WHOOP 4.0 wake-alarm
arming fix (#34), the menu tidy-up (#336), and the completed German translation (#326) - on top of
this fork's additions.

## Fork additions

- **Status-147 "strap not answering" recovery guidance (Android).** When a strap advertises but never
  answers connection requests (two consecutive establishment timeouts, GATT status 147 on Android 14+),
  NOOP now surfaces honest recovery steps - put the strap on its charger for a minute, toggle the
  phone's Bluetooth, restart the phone - instead of retrying silently forever. Deliberately distinct
  from the re-pair guides, because re-pairing does not fix this failure and costs a working pairing.
  Auto-reconnect keeps running; the guidance clears itself on the next successful connection.

## Notes

- Version numbering: this fork tracks upstream's version (8.7.0) and adds its own patch increment;
  build numbers continue the fork's own counters so sideloaded installs keep updating in place.
